# Personal Portfolio - Saatvik Gupta

A premium, highly interactive dark-mode portfolio website built using Next.js, TypeScript, Tailwind CSS, and Three.js.

## Tech Stack & Features
- **Frontend Core**: Next.js 15+ (App Router), TypeScript, React.
- **Styling**: Tailwind CSS, custom opaque dark glassmorphism card templates.
- **3D Graphics**: Vanilla Three.js rendering an interactive WebGL particle cloud in the Hero section.
- **Responsiveness**: Fluid layout adjustments across desktop and mobile screens.
- **Fonts**: Google Font Plus Jakarta Sans.

## Routes
- `/` - Landing page with conditional gate unmounting, Hero identity, full about me bio, experience timeline, and featured works.
- `/projects` - Complete portfolio showcase folder filterable by tags.
- `/skills` - Technical frameworks mapped to specialized category blocks.
- `/certifications` - Digital credential list complete with verification badge seals.
- `/contact` - Direct email action triggers and professional social links.

## Setup and Installation

Install dependencies:
```bash
npm install
```

Run dev server:
```bash
npm run dev
```

Build production static assets:
```bash
npm run build
```
